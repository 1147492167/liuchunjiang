IMG_9276 · 新能源汽车动力电池

核心考点速记

  • 三个原理：泡利不相容、能量最低、洪特规则
  • 四个量子数：n (主), l (角), m (磁), ms (自旋)
  • n型半导体：掺V族(P/As) → 电子为多子；p型：掺III族(B) → 空穴为多子
  • pn结：浓度梯度驱动扩散 → 复合 → 空间电荷区 → 内建电场 → 动态平衡
  • 四种极化：电子位移(10⁻¹⁵s) → 离子位移(10⁻¹³s) → 偶极子转向(10⁻⁶s) → 空间电荷(最慢)
  • 介电常数测量：C = ε₀εᵣA/d，通过测电容获得
  • 超导三指标：Tc (临界温度), Hc (临界磁场), Jc (临界电流密度)
  • 铁电性：自发极化 + 可随外电场反转；用电滞回线 (P-E) 表征
  • 锂电改进：包覆、掺杂、纳米化 → 提高电化学稳定性

IMG_9283 · 冲击法测量

核心考点速记

  • 五类磁性：抗磁(χ负小) → 顺磁(χ正小) → 反铁磁 → 亚铁磁 → 铁磁(χ正大)
  • 自发磁化必要条件：未填满电子壳层（未抵消自旋磁矩）；充分条件：交换积分 A > 0
  • 三种磁损耗：磁滞损耗 + 涡流损耗 + 剩余损耗
  • 降低损耗：高电阻率材料、粉末化、绝缘包覆、细化磁畴
  • 关键参数：Bs (饱和磁感应强度), Br (剩余磁感应强度), Hc (矫顽力)
  • VSM：振动样品磁强计，测量磁化曲线、磁滞回线、饱和磁化强度

IMG_9290 · 消光系数、复折射率

核心考点速记

  • 折射率 n = c/v：离子半径↑ → 易极化 → n↑；晶态 > 玻璃态
  • 复折射率 N = n - ik：k 为消光系数，k=0 无吸收
  • 禁带宽度 Eg：Eg 越小 → 吸收波长越长；宽禁带 → 可见光透明
  • 双折射：各向异性晶体不同晶轴方向 n 不同
  • 受拉应力：垂直方向 n 大，平行方向 n 小

核心考点速记
